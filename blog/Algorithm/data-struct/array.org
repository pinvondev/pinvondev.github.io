* 数组

在数组中, 相邻的两个元素, 物理地址也相邻.

* 向量

通过对数组进行抽象, 就得到了向量. 其抽象的内容在于, 同一向量中的各个元素可以不是同一个基本类型, 不能保证彼此之间可以比较大小.
