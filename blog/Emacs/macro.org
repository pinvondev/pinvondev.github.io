* 开始录制

C-x (

或者 F3

* 结束录制

C-x )

或者 F4

* 使用宏

C-u 0 C-x e

* 例子

按F3开始录制

输入: C-n C-a C-q Tab

按F4开始录制

C-u 0 C-x e: 执行宏, 将下面的代码使用Tab来调整.

#+BEGIN_SRC C++
int main(){
cout << "Hello" << endl;
cout << "World" << endl;
cout << "Test" << endl;
cout << "Emacs" << endl;
cout << "Macro" << endl;
return 0;
}
#+END_SRC
	
* 录制多个宏

F3 ... F4 后

输入 M-x name-last-kbd-macro

输入名字 test

使用: M-x test

* 重复执行次数

C-u 2 M-x test: 重复执行两次test宏
C-u 8 C-f: 前进8个字符
