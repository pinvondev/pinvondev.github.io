下载Scala

到官网下载

安装Scala

sudo tar -zxf scala-2.12.4.tgz -C /usr/local
cd /usr/local
sudo mv ./scala-2.12.4 ./scala
sudo chown -R pinvon:pinvon ./scala

emacs ~/.bashrc

export PATH=$PATH:/usr/local/scala/bin

source ~/.bashrc

# 测试
scala

使用

# 进入Scala环境
scala

scala> 1+1
res0: Int = 2

# 退出Scala环境
scala>:quit

编程

创建文件

cd /usr/local/scala
mkdir mycode
cd mycode
emacs test.scala

编写代码

object HelloWorld{
    def main(args: Array[String]){
        println("hello world")
    }
}

编译执行

scalac test.scala
scala -classpath . HelloWorld

注意，执行的时候使用的是 object 后面的名称，而不是文件名。
