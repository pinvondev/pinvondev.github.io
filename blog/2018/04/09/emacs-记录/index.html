安装

ubuntu 16.04 LTS

sudo add-apt-repository ppa:kelleyk/emacs
sudo apt update
sudo apt install emacs25

buffer大小调整

C-x { 扩大.

C-x } 缩小.

C-x - 缩小字体.

C-x + 放大字体.

C-x z 启动重复以上的命令, 再按 z 进行调整.
