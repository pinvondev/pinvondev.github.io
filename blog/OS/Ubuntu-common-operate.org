* tree 查看目录结构

#+BEGIN_SRC Shell
sudo apt install tree
tree -L 4 pathname # -L指定层级
#+END_SRC
* du 查看占用磁盘大小

du即disk usage

查看帮助:
#+BEGIN_SRC Shell
du --help
#+END_SRC

-h 表示结果以K, M, G为单位, 提高信息的可读性:
#+BEGIN_SRC Shell
du -h filename
#+END_SRC

如果只想看一个文件夹下所有文件的总和, 而不要看每个子文件的占用大小, 有两种办法, 一是使用-s参数, 二是指定最大深度为0
#+BEGIN_SRC Shell
du -sh filename

# 或

du -h --max-depth=0 filename
#+END_SRC
